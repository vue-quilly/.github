# VueQuilly

A simple, flexible Quill editor wrapper for Vue & Nuxt.

VueQuilly makes it easy to use Quill in your Vue apps without extra complexity or heavy abstractions.

## Why?

You just want a text editor that works — not a fight with configs.

VueQuilly gives you:
- Clean Vue 3 integration  
- Minimal setup 
- Full access to Quill instance when you need it  

## Features

- Dual Format Support (HTML and Quill Delta format)
- Lightweight and flexible  
- No styling lock-in  
- Nuxt support
- TypeScript support
